Cuisine Quest

A modern Recipe Discovery Web App built using React that allows users to explore, search, and view detailed recipes from different cuisines.

---

## 🌐 Live Demo

👉 [Live Project Link](https://cuisine-quest-dun.vercel.app/)
---

## 🚀 Features

- 🔍 Search recipes by cuisine
- 📂 Category-based filtering (Italian, American, etc.)
- 📄 Detailed recipe view (ingredients & instructions)
- ⚡ Fast and responsive UI
- 💾 Local storage caching for better performance
- 🎨 Clean and modern UI design

---

## 🛠️ Tech Stack

- Frontend: React, React Router
- Styling: Styled Components
- Animations: Framer Motion
- Carousel: Splide.js
- API: Spoonacular API
